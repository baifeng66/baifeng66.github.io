事件绑定

事件修饰符 once –> 代表只能执行一次

例如：v-on:click.once=”upd”,该点击事件执行一次后就没效果了

v-for循环

用于循环渲染列表或数组的元素,Vue要求为每个循环的元素提供一个唯一的key属性,key属性的作用是帮助Vue跟踪每个循环的元素，以便在列表发生变化时进行高效的更新。

v-if 判断

组件选项

  • data选项：存放数据
  • methods选项：存放方法
  • computed选项：存放需要重复计算的数据（通过缓存方式提高性能）

Vue-Cli

主要功能

  1. 转译(ES6-ES11 高版本代码–>低版本代码) – (Babel)
  2. 代码质量(ESlint)
  3. 单文件组件(SFC)–> 组件= template+JS+CSS
  4. 代码压缩

安装以及使用

  • npm install -g @vue/cli (全局安装 vue-cli工具)
  • vue create ant (创建一个名字为 ant 的vue项目)

组件由三个主要部分组成：模板（template）、脚本（script）和样式（style）。
