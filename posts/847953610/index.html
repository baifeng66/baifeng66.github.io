Java基础之路-动力节点

洋洋洋2023-10-172023-10-28

基础

语言特性

  • 简单性
    • 不支持多继承，没有指针的概念，底层由c++实现
  • 面向对象
  • 可移植性
    • 一次编译，到处运行
    • 通过 jvm实现
  • 多线程
  • 健壮性
    • 自动回收垃圾机制(GC)
  • 安全性

加载与执行

编译阶段

  1. 检查Java源代码是否符合Java语法，符合则生成字节码文件(xxx.class)

  2. 字节码文件无法直接在操作系统中执行

  3. 编译命令：javac ../../..../xxx.java

  4. 一个源文件可以生成多个字节码文件

运行阶段

  1. 运行命令: java ../../.../xxx(类名)
  2. java命令会启动Java虚拟机(jvm)，jvm会启动类加载器ClassLoader
  3. ClassLoader回去硬盘搜索xxx.class文件，找到并且将其装载到jvm中
  4. jvm将xxx.class字节码文件解释成二进制数据(1010010101…)
  5. 操作系统执行二进制和底层硬件平台进行交互

JDK JRE JVM 关系

  1. JDK是用于开发Java应用程序的工具包，包括了JRE

  2. JRE是用于运行Java应用程序的运行时环境，包括了JVM和Java类库

  3. 而JVM是Java应用程序运行的核心，负责解释和执行Java字节码

环境变量

  1. 使用终端：您可以使用终端来打开环境变量文件。

  2. 加入java环境

Classpath用于告诉JVM在哪里查找Java类文件，而Path用于告诉操作系统在哪里查找Java可执行文件。设置正确的Classpath和Path对于编译和运行Java程序非常重要，以确保JVM能够找到所需的类文件和可执行文件。

  2. /** javadoc注释

public class 和 class的区别

  1. 一个java源文件当中可以定义多个class
  2. 一个java源文件当中public的class不是必须的
  3. 一个class会定义生成一个xxx.class字节码文件
  4. 一个java源文件当中定义公开的类的话，只能有个，并且该类名称必须和java源文件名称一致
  5. 每一个class当中都可以编写main方法，都可以设程序的入口，想执行B.class中的main方法:java B，想执行x.class当中的main方法:java x
  6. 注意:当在命令窗口中执行java hello，”那么要求hello.class当中必须有主方法。没有主方法会出现运行阶段的错误

标识符

用来命名变量，方法，类名，接口名…程序元素的名称

命名规则

  • 只可以包含字母（A-Z，a-z）、数字（0-9）、下划线（_）和美元符号（$）
  • 必须以字母、下划线、美元符号开头，不能以数字开头
  • 严格区分大小写
  • 理论上无长度限制
  • 不能为Java的关键字(class,public,int…这种单词)

命名规范

  • 使用有意义的名称
  • 遵循驼峰命名法
    • 变量名、方法名：首字母小写，后面每个单词首字母大写
    • 类名、接口名：首字母以及后面每个单词的首字母都大写
    • 常量名：全部大写

字面值

字面值是直接出现在代码中的常量值，它们表示了不同类型的数据，如整数、浮点数、字符、字符串、布尔值等。

变量

变量是用来存储和表示数据的内存位置(空间)。每个变量都有一个特定的类型，它限制了变量可以存储的数据类型和范围

语法

数据类型 变量名 = 字面值;

  • 必须先声明，再赋值(也可同时操作)
  • 可一行多个变量赋值
    • int a=1,b=2,c=3;
  • 在同一作用域中，变量名不能重名
    • 变量的作用域是指变量在程序中可见和可访问的范围。变量的作用域由变量的声明位置决定
    • 局部变量（Local Variables）：局部变量是在方法、构造函数或代码块内部声明的变量。
      • 它们只在声明它们的方法、构造函数或代码块中可见。局部变量在声明时需要初始化，并且只在其声明的范围内有效
    • 实例(成员)变量（Instance Variables）：实例变量是在类中声明的变量，但在方法之外。
      • 它们属于类的实例，每个类的实例都有自己的一组实例变量。实例变量在类的任何非静态方法中都可以访问。
    • 静态变量（Static Variables）：静态变量是使用 static 关键字在类中声明的变量。
      • 它们属于类本身，而不是类的实例。静态变量在类的任何方法中都可以访问，甚至在没有创建类的实例的情况下也可以访问。
    • 方法参数（Method Parameters）：方法参数是在方法声明中定义的变量。它们在方法被调用时接收传递给方法的值。
      • 方法参数的作用域仅限于方法内部。

数据类型

数据类型用于定义变量的类型和范围，以及变量可以存储的数据类型。Java提供了两种数据类型：基本数据类型和引用数据类型。

基本数据类型

  • 整数类型

  • 默认值：0

    • byte 1字节
    • short 2字节
    • int 4字节
    • long 8字节

  • 浮点型

    • 默认值：0.0
    • float 4字节
    • double 8字节

  • 布尔型

    • 默认值：false
    • boolean 1字节

  • 字符型

    • 默认值：’/u000’
    • char 2字节

计算机在任何情况下只认识二进制。

二进制是一种计数系统，只使用两个数字 0 和 1 来表示数值。数字类型有正负数之分，二进制最左边的数字为符号位。0代表正数，1代表负数。

字节(byte)：

  • 1 Byte = 8 bit(位)
  • 1 KB = 1024 Byte
  • 1 MB = 1024 KB
  • 1 GB = 1024 MB
  • 1 TB = 1024 GB
